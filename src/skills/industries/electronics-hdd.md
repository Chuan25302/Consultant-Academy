# Electronics — HDD / Semi Packaging — Skill Card

## Thai context
- Thailand = world leader in HDD: Western Digital (Bang Pa-in), Seagate (Korat, Teparuk)
- Semi packaging (ATP/OSAT): Microchip, Hana Microelectronics, KCE, SVI
- PCB: Delta Electronics, Cal-Comp
- Cluster: Bang Pa-in, Korat, Lamphun (Northern), Bangpoo

## Energy profile (typical HDD plant)
- **Clean room HVAC**: 50–70% of plant energy (recirculation + makeup air + RH control)
- Clean room class 100/1000 (or ISO 5/6) demands 60–600 air changes/hour
- Process tools + chillers (precision ±0.5°C cooling): 15–25%
- Compressed dry air (CDA) + N2: 10–15% (very dry, very clean)
- Lighting + general: 5–10%

## High-impact opportunities
- Clean room recirculation VSD on FFU (Fan Filter Units): 15–30% HVAC savings
- Free cooling economizer (limited in Thailand climate but possible at night)
- Reduce clean room class where process allows (re-classification project)
- Heat recovery from process chillers
- LED in clean rooms (use clean-room-rated fixtures)

## Standards
- **IPC** standards (assembly, soldering)
- **ESD S20.20** — ESD control (mandatory for electronics)
- **ISO 14644** — clean room classification
- **IATF 16949** — for automotive electronics suppliers

## Talking to this customer
- Facility Engineer + Process Engineer + Safety/Environment
- Customer-driven (Apple/HP/Dell supplier programs) → Scope 3 pressure
- Production uptime is sacred — energy projects must not risk output
- Budget cycles tied to fiscal year of parent (often US/Japan)
