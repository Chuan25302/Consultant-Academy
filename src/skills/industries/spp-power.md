# SPP / VSPP / Biomass / Biogas / Cogen — Skill Card

## Thai context
- **SPP** (Small Power Producer): 10–90 MW, sells to EGAT under PPA
- **VSPP** (Very Small Power Producer): <10 MW, distribution-level
- **Cogeneration**: gas turbine + HRSG + steam turbine (combined cycle)
- Biomass: rice husk, bagasse (sugar mill waste), wood chip
- Biogas: pig farm, palm oil mill effluent (POME), MSW landfill

## Common configurations in Thailand
- **SPP cogen** (most common in petrochem/industrial estates):
  Gas turbine 25–60 MW + HRSG → steam turbine 10–30 MW → process steam
- **Biomass-fired**: stoker grate or fluidized bed, 5–25 MW typical
- **Biogas-to-power**: gen set on pre-treated biogas, 0.5–5 MW
- **VSPP solar**: PV + inverter, sometimes with BESS

## Key efficiency metrics
- **Heat rate** (kJ/kWh) — lower is better
- **Capacity factor** = actual output / nameplate (target >70% baseload)
- **Thermal efficiency** = useful output / fuel energy
- For cogen: also track power-to-heat ratio + total system efficiency

## High-impact opportunities
- HRSG fouling cleanup + steam quality
- Air preheater on boiler (5–8% efficiency gain)
- Compressor inlet air cooling (gas turbine — 0.5–1% per °C reduction)
- Biomass fuel handling + drying (wet fuel kills efficiency)
- O&M optimization (often best practice gap is large)

## Standards / Regulatory
- **ERC** (Energy Regulatory Commission) — license + tariff
- **ASME I/IV** — boilers + pressure vessels
- **IEEE / Grid Code Thailand** — interconnection
- **EHIA** — for >10 MW or specific fuel types
- **TGO + T-VER** — biomass/biogas projects can register for credits

## Talking to this customer
- Plant Manager + O&M Manager + Commercial Manager (PPA + fuel contract)
- They live in P&L of MWh sold vs fuel cost — every 0.1% efficiency matters
- Carbon credits add a real revenue stream for biomass/biogas projects
