# Motor — Skill Card

## Key concepts
- **IE class**: IE1 (standard), IE2 (high), IE3 (premium), IE4 (super premium)
- **Slip** = (synchronous − actual) ÷ synchronous; small motors have higher slip %
- **Soft starter vs VFD**: VFD = variable speed + soft start + harmonics filter
- **Power factor**: typically 0.85–0.95 at full load, drops sharply at part load
- **Service factor**: short-term overload tolerance (e.g., 1.15)

## Common issues
- Oversized motors running at 30–50% load — efficiency tanks below ~50% load
- Failed bearings → vibration, heat, losses (replace bearings every 3–5 yrs typical)
- V-belt slip vs direct drive vs synchronous belt
- Dust accumulation on motor cooling fins

## Retrofit options + ROI
- IE2 → IE3: ~3–4% efficiency gain (worth it on continuous loads >4000 hr/yr)
- Add VFD on variable load (pumps/fans/compressors): 10–30% savings
- Right-size: replace 30 kW running at 12 kW with 15 kW unit → savings 5–15%
- Direct drive replacing V-belt: 1–3% savings

## Standards
- **TIS 2780** — motor efficiency, IE3 mandatory for select sizes (Thailand)
- **IEC 60034-30-1** — efficiency classes
